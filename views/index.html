Simple Inventory API

Ласкаво просимо до сервісу обліку обладнання.

  • GET /items - Список усіх предметів (HTML/JSON)
  • POST /items (name, quantity) - Створити новий запис
  • GET /items/<id> - Деталі предмету (HTML/JSON)

Health Checks:

  • GET /health/alive - Перевірка доступності сервісу
  • GET /health/ready - Перевірка підключення до бази даних
